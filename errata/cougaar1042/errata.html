Errata for Cougaar 10.4.2


The following infos/problems/workarounds were detected/posted after Cougaar 10.4.2 was released.

Issues:

  • none identified

New Bugs:

  • There is a deadlock when Quiescence reports are turned on. These are off by default with the Cougaar 10.4.2 distribution. This is Cougaar bug 3168.
  • There is a ConcurrentModificationException in the CommunityManager that occurs infrequently (I've seen it once in about 20 runs). This is Cougaar bug 3155.
  • Freezing the society doesn't work properly. The freeze servlet says "Agent Freezing", but never is updated to say that it has completed the freeze. This is Cougaar bug 3170.
