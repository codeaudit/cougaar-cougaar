Errata for Cougaar 10.0.0 (also known as 10.0)


The following infos/problems/workarounds were detected/posted after Cougaar 10.0.0 was released.

Issues:

  • none so far

New Bugs:

  • YP fails to work as advertised due to missing code in yp/WaitQueue (bug 2562). Fixed for 10.2 and in all post 10.0 patch releases. If you would like to try out the YP package, you will need a patch: please contact cougaar-developers for more information.
